# Local File Integrity Checker

## 📌 Objective
To detect file tampering using SHA-256 hashing.

## 🚀 Features
- Select file from system
- Generate SHA-256 hash
- Detect file modification
- Show original vs current hash
- Clean GUI interface (Tkinter)

## 🛠️ Technologies Used
- Python
- Tkinter
- Hashlib

## ▶️ How to Run
1. Run `integrity_ui.py`
2. Select a file
3. Click "Check Integrity"
4. Modify file and check again
